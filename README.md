# MongoDB & Mongoose CRUD Application

A simple Node.js application demonstrating how to perform basic **CRUD (Create, Read, Update, Delete)** operations using **MongoDB Atlas** and **Mongoose**.

This project serves as a foundational exercise for understanding how backend systems interact with databases in real-world applications.

---

## Features

- Connect to MongoDB Atlas using Mongoose
- Create and save a document
- Create multiple records
- Find documents (by name, by ID, by field)
- Update documents
- Delete single and multiple records
- Chain query helpers for advanced filtering

---

## Technologies Used

- Node.js
- MongoDB Atlas
- Mongoose
- dotenv
